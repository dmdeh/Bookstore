# Bookstore

## 요구사항

### 📄 책 목록 페이지

- [ ] 책 목록을 가져와 10개씩 페이지네이션
- [ ] 제목과 저자로 검색 기능 구현

### 📄 책 상세 정보 페이지

- [ ] 선택한 책의 상세 정보 표시

### 📄 CRUD 기능

- [ ] 책 추가
- [ ] 책 삭제
- [ ] 책 정보 수정 (제목, 저자, 수량 등)
  - [ ] 책 수량 증가/감소 조절 기능

### 📌 RESTful API

- [ ] 책 목록 조회 `GET /api/books`
- [ ] 책 상세 조회 `GET /api/books/:id`
- [ ] 책 추가 `POST /api/books`
- [ ] 책 정보 수정 `PUT /api/books/:id`
- [ ] 책 삭제 `DELETE /api/books/:id`
